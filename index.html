USER GUIDE

Jeevadhaar is a mobile application which is used to create a database of all the stray animals (dogs, cow, cows, donkeys, etc) in a city.

The User shall make an account, then they'll be able to access many features. Few main features are:- accessing detals of a specific stray animal , creating new entry for a new animal, see and contact animal helplines, manage your info, edit database, see latest updates on animal rescues, access user guide, and educate yourself.



We're using the following technologies and frameworks


    -Flutter (for developing the mobile application)
    -Android Studio (for writing and editing our Flutter code)
    -Firebase Firestore (for login authentication and database management)
    -HTML-CSS (for creating various webpages that are being integrated in the app)
    -Atom Editor (for writing and editing our HTML-CSS code)
    -PicsArt (for making graphical illustrations and logo)
    -GitHub (for making respositories of our app and hosting our website)
